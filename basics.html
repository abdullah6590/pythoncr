1. Introduction and First Program

2. Python Syntax and Indentation

Python uses indentation (spaces) to define code blocks instead of braces.

  • Same block = same indentation level.
  • Default is 4 spaces in most editors.

Task 2: Fixing Indentation

Add one more print inside the if block. Then try removing the indentation and see what happens!

if 8 > 2: print("8 is greater than 2") print("This line is in the same block") if 10 > 3: print("10 is greater than 3") print("I added a new line properly indented!")

3. Comments

Comments are notes for humans, ignored by Python.

  • Single-line: starts with #
  • Multi-line: multiple # lines or triple quotes """ ... """

Task 3: Leaving Notes

Write a short program with at least 3 comments (e.g., Print your name and age).

# This program introduces me # I'll print my name first print("My name is Ali") # Now I will print my age print("I am 22 years old")

4. Variables

A variable stores a value with a name.

  • Use = to assign.
  • Names: letters, numbers, underscore; cannot start with a number.
  • Case-sensitive: age and Age are different.

Task 4: Creating Your Data

Create 3 variables: your_name, your_city, your_year_of_birth. Then print them in one sentence.

your_name = "Abdullah" your_city = "Peshawar" your_year_of_birth = 2000 print("I am", your_name, "from", your_city, "born in", your_year_of_birth)

5. Data Types and Casting

Common built-in types: str, int, float, bool.

Casting = converting between types using int(), float(), str().

Task 5: Text to Number Magic

Ask the user for a number (as text), convert to int, and add 10.

text_number = input("Enter a number: ") # Convert to integer and add 10 result = int(text_number) + 10 print(f"Your number plus 10 is: {result}")

6. Strings

Strings are text inside quotes. Key operations:

  • Indexing: s[0]
  • Slicing: s[0:5]
  • Methods: upper(), lower(), strip(), replace(), split()
  • Concatenation: "Hello " + "World"
  • f-strings: f"Hello {name}"

Task 6: String Manipulation

Given a name string, print: (1) Uppercase (2) First 3 characters (3) A sentence: "Hello, <name>!"

student_name = "Abdullah" print(student_name.upper()) print(student_name[0:3]) print(f"Hello, {student_name}!")

7. Booleans and Comparisons

A boolean is either True or False.

  • Comparison operators: ==, !=, >, <, >=, <=
  • Logical operators: and, or, not

Task 7: Truth Teller

Create two numbers. Print True if both are greater than 5, otherwise False.

x = 7 y = 2 if x > 5 and y > 5: print(True) else: print(False)

8. Operators

Types of operators:

  • Arithmetic: + - * / % // **
  • Assignment: = += -= *=
  • Comparison: == != > <
  • Logical: and or not
  • Membership: in, not in

Task 8: The Calculator

a = 15 b = 4 print("Add:", a + b) print("Subtract:", a - b) print("Multiply:", a * b) print("Divide:", a / b) print("Floor Divide:", a // b) print("Modulo (Remainder):", a % b) print("Power:", a ** b)

9. Lists

Lists store multiple items in order and are mutable (can change).

Important methods: append(), insert(), remove(), pop()

Task 9: Fruit Basket

Try modifying a list by appending and removing items.

10. Tuples

Tuples are like lists but immutable (cannot change items after creation).

Syntax: my_tuple = (1, 2, 3)

Task 10: Favorite Foods

Create a tuple of 4 favorite foods. Print each item using a loop.

11. Sets

Sets are unordered collections of unique items. They instantly remove duplicates!

Syntax: {1, 2, 3}

Task 11: Common Students

Create two sets of student IDs. Print the IDs that are in both sets using intersection (&).

class_a = {1, 2, 3} class_b = {3, 4, 5} common_students = class_a & class_b print("Students in both classes:", common_students)

13. If, Elif, Else

Conditional statements let the program make decisions based on rules you define.

Task 13: Age Checker

Write a program that uses an age variable to print "Child", "Teenager", or "Adult".

14. While Loops

A while loop repeats while a condition is True. Be careful to avoid infinite loops!

Think of it like a traffic light: "I will wait while the light is red." You don't know exactly how long it will take, but you know the condition.

üö® The Danger Zone: Infinite Loops
If you forget the count += 1 line, the computer will never stop counting because the condition never becomes False! If you do this, your program will freeze!

Task 14: The Countdown

Print numbers from 10 down to 1 using a while loop.

count = 10 while count > 0: print(count) count -= 1 # DON'T FORGET THIS STEP! print("Blastoff!")

15. For Loops

for loops iterate over sequences (like lists, strings, or numbers safely ending automatically).

Task 15: The Roll Call

Loop over a list of 5 names and print "Hello, <name>".

names = ["Ali", "Sara", "Bilal", "Noor", "Zain"] for name in names: print(f"Hello, {name}!")
