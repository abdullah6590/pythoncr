Python Mastery

From Variables to JSON: An Interactive Infographic Guide

Variables #01
A named container that stores a value in memory. Created with =, type decided by value.
Easy
name = "Ali"          
age = 20              
is_student = True     
print(name)
Moderate (Dynamic Typing)
x = 10                
x = "Now I am text"   
print("x is:", x)
Expert (Swap)
a, b, c = 1, 2, 3     
a, b = b, a           
print(a, b)  # 2 1
Tricky
a = 5                 
b = a                 
a = a + 3             
c = b                 
b = a                 
a = c                 
# a=5, b=8, c=5
Data Types #02
Describes the kind of value a variable holds (int, float, str, bool, list, etc.).
Easy
x = 10                # int
y = 3.14              # float
z = "Python"          # str
flag = False          # bool
Moderate (Mixing Types)
a = 5
b = 2.0
result = a + b        
# 7.0 (float)
Expert (Mixed List)
mixed = [1, "two", 3.0]
for item in mixed:
    print(type(item))
Tricky
x = "5"
x = x * 2            
print(x)    
# "55" (string repetition)
Casting #03
Converting a value from one data type to another (e.g. str to int).
Easy
x = "3"
y = int(x)   # 3
z = float(x) # 3.0
Moderate
num = 10
text = str(num)
msg = "Nr is " + text
# "Nr is 10"
Expert (Safe Cast)
try:
    val = int("abc")
except ValueError:
    val = None
Tricky
try:
    b = int("7.5") # Fail
except:
    b = float("7.5")
# 7.5 (float)
Strings #04
A sequence of characters used to represent text.
Easy
text = "Hello"
print(text[0])  # 'H'
print(text[-1]) # 'o'
Moderate
s = "  hello  "
print(s.strip().upper())
# "HELLO"
Expert (f-strings)
name, age = "Ali", 21
msg = f"I am {name}, {age}"
# "I am Ali, 21"
Tricky (Slicing)
s = "python"
print(s[1:4])  # 'yth'
print(s[::-1]) # 'nohtyp'
print(s[::2])  # 'pto'
Booleans #05
A value that is either True or False.
Easy
a = True
b = False
print(a, b)
Moderate
x, y = 10, 5
print(x > y)  # True
print(x == y) # False
Expert (Truthiness)
bool(0)     # False
bool("")    # False
bool("0")   # True
Tricky
x, y = 0, 5
if x and y: ...
elif x or y:
    print("One is True")
# Output: One is True
Operators #06
Symbols that perform operations on variables and values (+, -, *, /, etc.).
Easy
x = 7
y = 3
print(x / y) # 2.333...
Moderate
print(7 // 3) # 2 (floor)
print(7 % 3)  # 1 (rem)
print(7 ** 3) # 343 (pow)
Expert (In-place)
a = 5
a += 2  # 7
a *= 3  # 21
Tricky
res = 10
res /= 2   # 5.0
res //= 2  # 2.0
res += 3   # 5.0
Lists #07
An ordered, mutable collection of items.
Easy
f = ["apple", "banana"]
print(f[0])   # apple
print(len(f)) # 2
Moderate
f.append("orange")
f.insert(1, "kiwi")
# apple, kiwi, banana...
Expert (Comprehension)
nums = [1, 2, 3]
sq = [n*n for n in nums]
# [1, 4, 9]
Tricky (Reference)
a = [1, 2]
b = a
b.append(3)
# a is now [1, 2, 3]
Tuples #08
An ordered, immutable collection of items.
Easy
t = (1, 2, 3)
print(t[0]) # 1
Moderate
for item in (1, "A"):
    print(item)
Expert (Unpacking)
data = ("Ali", 21)
name, age = data
# name="Ali", age=21
Tricky (Mutable Inner)
t = (1, [3, 4])
t[1].append(5)
# (1, [3, 4, 5])
# Valid: list is mutable.
Sets #09
An unordered collection of unique items.
Easy
nums = {1, 2, 3, 3}
# {1, 2, 3} (unique)
Moderate
a = {1, 2}
b = {2, 3}
print(a | b) # Union {1,2,3}
print(a & b) # Inter {2}
Expert
print(1 in {1, 2}) # True
Tricky
s = set()
s.add(1)
s.add(1)
print(len(s)) # 1

For Loops #11
Repeats code for each item in a sequence.
Easy
for i in [1, 2, 3]:
    print(i)
Moderate
for char in "Hi":
    print(char)
Expert (Loop Else)
for i in range(3):
    print(i)
else:
    print("Done")
Tricky
for i in range(5):
    if i == 3: break
else:
    print("Finished")
# 'else' skipped due to break
While Loops #12
Repeats code as long as the condition is True.
Easy
i = 1
while i <= 3:
    print(i)
    i += 1
Moderate
count = 3
while count > 0:
    count -= 1
Expert (Break)
while True:
    if i >= 3: break
Tricky
i = 0
while i < 5:
    i += 2
    print(i)
# 2, 4, 6
Functions #13
A reusable block of code that runs when called.
Easy
def greet():
    print("Hello!")
greet()
Moderate
def add(a, b):
    return a + b
print(add(3, 4)) # 7
Expert (Defaults)
def info(name, age=18):
    print(name, age)
info("Ali") # Ali 18
Tricky (Mutable Default)
def add(x, l=[]):
    l.append(x)
    return l
# l is shared across calls!
print(add(1)) # [1]
print(add(2)) # [1, 2]

Data Handling #17
Using collections like lists/dictionaries to store and process info.
Easy

Expert

Math #18

Easy

Moderate

JSON #19
Text format for data exchange. Python converts to/from dicts/lists.
Easy
import json
p = {"name": "Ali"}
js = json.dumps(p)
Moderate
obj = json.loads('{"y":1}')
print(obj["y"]) # 1
Expert (Pretty)
print(json.dumps(p, indent=4))
Tricky
load = json.loads('[1, 2]')
print(type(load))
