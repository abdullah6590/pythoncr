1. Variables

Named containers storing values in memory.

EASY
name = "Ali"
age = 20
print(name, age)
MODERATE
x = 10 # int
x = "Text" # str
print(x)
EXPERT
a, b = 1, 2
a, b = b, a # Swap
print(a, b) # 2 1
TRICKY

Predict a,b,c:

a=5; b=a; a=a+3; c=b; b=a; a=c
Reveal
a=5, b=8, c=5

2. Data Types
