🎲 Probability in Machine Learning

The mathematics of uncertainty. Because in the real world (and in ML), absolute certainty is an illusion.

In Machine Learning, our predictions are rarely exact. Models make guesses based on data. Probability helps us model this uncertainty and make intelligent, calculated decisions instead of blind guesses!

1️⃣ 2️⃣ Introduction & Terminology

Why Probability is Important in ML

  • Classification problems: "Is this email 90% likely to be spam?"
  • Prediction under uncertainty: Dealing with noisy or incomplete real-world data.
  • Bayesian models: Algorithms completely driven by probability.
  • Neural network outputs: Softmax and sigmoid functions output probabilities!

The Foundation Words

S = {H, T}

3️⃣ 4️⃣ Definition & Random Variables

The Definition & Properties

P(A) = Favorable Outcomes / Total Outcomes

  • 0 ≤ P(A) ≤ 1 (Probability is between 0% and 100%)
  • P(S) = 1 (Something in the sample space WILL happen)
  • P(∅) = 0 (Impossible events have 0 probability)

Random Variables

A random variable assigns numerical values to outcomes. Because equations need numbers, not words!

Types:
  • Discrete: Countable values (e.g., shoe size, coin tosses).
  • Continuous: Infinite possible values (e.g., height, temperature).
Example: X = Number of heads in a coin toss.

5️⃣ Probability Distributions

A probability distribution is just a mathematical function that gives the probabilities of occurrence of different possible outcomes. It shows how the data is "distributed"!

Discrete Distributions (Categorical / Countable)

🪙 Bernoulli Distribution

A single trial with only two possible outcomes (Success/Failure).

P(X = 1) = p, P(X = 0) = 1 − p
Used in ML: Binary classification (e.g., Spam vs Not Spam where output is 0 or 1).

🎲 Binomial Distribution

The number of successes in n repeated independent Bernoulli trials.

P(X = k) = C(n,k) p^k (1−p)^{n−k}
Used in ML: Repeated experiments, A/B testing variations.

Continuous Distributions (Measured / Infinite)

📏 Uniform Distribution

All values within a certain continuous range are equally likely.

Imagine a flat horizontal line graph.

🔔 Normal (Gaussian) Distribution

The famous "Bell Curve". Most values cluster around the mean.

f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
Used in ML: Modeling noise in data, core assumption in many ML algorithms!

6️⃣ 7️⃣ 8️⃣ Event Relationships

🤝

Joint Probability

The probability of two events happening together.

P(A, B) = P(A ∩ B)
"What's the probability of it raining AND being a Tuesday?"
🕵️

Conditional Probability

Probability of A given that B has already occurred.

P(A|B) = \frac{P(A ∩ B)}{P(B)}
"Probability of lung cancer GIVEN the person is a smoker."
🆓

Independence

Events do not affect each other. Knowing B happened gives no info about A.

P(A ∩ B) = P(A)P(B)
Like flipping a coin and rolling a die.

9️⃣ Bayes Theorem VERY IMPORTANT 🔥

Bayes' Theorem provides a way to update our belief when we see new data. It is the core of "Bayesian Inference". It flips conditional probabilities!
P(A | B) = \frac{P(B | A) P(A)}{P(B)}
P(A|B) Posterior: Our updated belief of A after seeing B.
P(A) Prior: Our initial belief of A before seeing the data.
P(B|A) Likelihood: Probability of observing B if A is true.
P(B) Evidence: Total probability of the new data B.

🤖 Application in ML:

Naive Bayes Classifier: Determines the probability an email is spam given the words inside it. Used heavily in Spam Filtering and text categorization.

🔟 1️⃣1️⃣ 1️⃣2️⃣ Expectation & Maximum Likelihood

⚖️ Expectation (Expected Value)

The average value of a random variable over many trials.

E[X] = Σ xP(x)
Use in ML: Used to define Loss Functions (we want to minimize the 'expected' error) and make average predictions.

🔍 Likelihood vs Probability

  • Probability: P(Data|Model)
    Predicting data given known model parameters.
  • Likelihood: L(Model|Data)
    Evaluating the model given the observed data.
Importance: Crucial for parameter estimation and model training.

🏆 Maximum Likelihood Estimation (MLE)

The goal of MLE is to find the parameters (θ) that best explain the observed data. It maximizes the likelihood function!

θ* = arg max P(Data | θ)

Example: If 8 out of 10 coin flips are heads, MLE says the probability of heads shouldn't be 0.5, but rather 0.8 (the value that maximizes the likelihood of seeing what we just saw).

1️⃣3️⃣ 1️⃣4️⃣ Entropy & Cross Entropy

🌀

Entropy (Uncertainty)

A measure of uncertainty or randomness in a system.

H(X) = - Σ P(x)\log_2 P(x)
Use in ML: Core to building Decision Trees (we split nodes to minimize entropy/maximize information gain).
❌

Cross Entropy

Measures the difference between two probability distributions (the true labels vs predicted probabilities).

L = - Σ y \log(ŷ)
Use in ML: The most common loss function for classification neural networks.

🧠 15.3 Bayes Theorem

P_A = 0.01 # Prior P(Disease) P_B_given_A = 0.9 # Likelihood P(Positive Test | Disease) P_B = 0.05 # Evidence P(Positive Test) # Posterior: P(Disease | Positive Test) P_A_given_B = (P_B_given_A * P_A) / P_B print(f"Updated belief (Posterior): {P_A_given_B}")

🌀 15.4 Entropy

import numpy as np # 50/50 Chance (Maximum uncertainty) p = [0.5, 0.5] # Calculate Entropy entropy = -sum([pi * np.log2(pi) for pi in p]) print(f"Entropy: {entropy} bits")

⚠️ 17. Common Mistakes

  • Confusing conditional probability P(A|B) with P(B|A) (Thinking the chance of being a smoker given lung cancer is the same as getting lung cancer given you're a smoker).
  • Misunderstanding independence (Assuming two features are independent when they actually influence each other).
  • Ignoring distributions (Applying a model that expects normally distributed data to heavily skewed data).
  • Mixing probability with purely statistics concepts.

📝 18. Summary

Probability is the foundation of intelligent decision-making in machine learning.

  • Probability handles uncertainty.
  • Distributions model how data looks.
  • Bayes updates our belief based on new evidence.
  • MLE estimates parameters.
  • Entropy measures uncertainty/randomness.
