üìñ What is a Function? (The Magical Pizza Box)

Imagine you have a magical box in your kitchen. If you put raw ingredients into a slot on top of the box, it magically bakes them and pops out a hot, delicious pizza from the bottom.

In Python, a Function is exactly like that magical box! Here are the secret words to build one:

  • def : This is the magic password. It stands for "define". It tells Python, "Hey! I am building a new magical box!"
  • Function Name : This is what you name your box. (e.g., make_pizza). It should describe what the box does.
  • () Parentheses : These are the slots on top of the box where you drop your ingredients.
  • Parameters : The labels on the slots. It tells us what ingredients the box expects (e.g., cheese, dough).
  • Arguments : The ACTUAL real ingredients you put in the slots later when you use the box (e.g., "Mozzarella").
  • Body : The indented code inside. This is the recipe. It tells the box exactly how to mix the ingredients together.
  • return : The door at the bottom of the box. This sends the finished, baked result back out to you so you can eat it!

To use the box later: You just call its name and give it ingredients! Like this: make_pizza("Mozzarella").
